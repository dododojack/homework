課程心得

課程初衷與期待

在這門課剛開始時，我對於網站開發還不是很熟悉。雖然以前接觸過一些 HTML，但還沒真正了解如何把整個網站結合樣式與邏輯。

學習歷程

過程中學到很多實用的內容，例如 HTML 的基本結構、CSS 的排版技巧，還有 JavaScript 的互動操作。這些知識讓我開始能夠獨立完成一些簡單的作品。

尤其是做作業時，從無到有看到自己的作品逐漸成型，真的非常有成就感。

遇到的挑戰與突破

有時候排版遇到瓶頸，例如圖片跑位或版面跑掉，會讓我很困擾。不過透過查資料、請教同學老師，慢慢都能解決，也讓我更熟悉排版邏輯。

未來展望

這門課激起我對網頁開發的興趣，未來我希望能學更多進階技術，例如 RWD 響應式設計或前端框架（像是 React）。也會繼續做 side project 累積實力。
